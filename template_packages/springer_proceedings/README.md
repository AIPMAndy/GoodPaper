# Springer Proceedings Package

This folder contains a sample manifest for a Springer-style package.

## Important

The actual template file is **not committed** in the open-source repository.
If you want to use this package locally, place your own Springer-compatible template file at the path referenced by `manifest.json`, or update the manifest to point to your local template.

## Intended use

- title
- author
- abstract
- keywords
- headings
- references

This package is kept mainly as a reference example for how to define a template package.
